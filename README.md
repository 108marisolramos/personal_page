# Personal Page — Marisol Ramos Correa

Personal portfolio built with HTML, CSS and Bootstrap 5. The contact form is a React component.

## Files

- `index.html` — main page (about, experience, education, skills, languages)
- `form.html` — contact page (React form)
- `styles.css` — custom styles
- `images/` — profile photo

## Run locally

```bash
python3 -m http.server 8000
# open http://localhost:8000
```
